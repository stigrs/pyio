# PYIO

Provides a collection of Python I/O utilities.

## Licensing

PYIO is released under the [MIT](LICENSE) license.

## Obtain the Code

The source code can be ob

    git clone git@github.com:stigrs/pyio.git

## Requirements

* [Python](https://docs.python.org/3/) 3.6

## Installation

The program is install by

    python setup.py install

Testing of the program can be done by:

    pytest
